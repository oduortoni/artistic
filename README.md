# Artistic

## Description

A program to perfoem graphics functions in the terminal while using pure golang standard library. It utilizes math to print to screeen.
 It is a project that I have a passion for since at times you just want to visualize something using a simple library. 
 During this times, having to utilize a large library is not ideal.

 # Usage

 ```bash
    git clone https://github.com/oduortoni/artistic.git
```

run the program
```bash
    cd artistic/cmd
    go run .
 ```

 alternatively, you can (while in the root folder):

 ```bash
   make run
```
